## How to install

- Unity -> Window -> Package Manager
- Click "+" at the top left corner
- Add package from git URL
- Insert `https://github.com/aether4co/com.aetherforgeai.sprite-shader.git`
- Add
- Done

## How to use

- Use the Remove Background From Aether Sprite Shader Graph in the Shader folder to remove backgrounds.
- For detailed usage, refer to the Sample.
